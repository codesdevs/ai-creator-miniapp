# uniapp-mini

这里预留为微信小程序端工程目录。

推荐技术栈：

- `uni-app`
- `Vue3`
- `Pinia`
- `uview-plus` 或 `uv-ui`

建议页面结构：

- `pages/app/index`
- `pages/home/index`
- `pages/assets/index`
- `pages/works/index`
- `pages/mine/index`
- `pages/create/image`
- `pages/create/video`
- `pages/order/recharge`

后续我们可以直接在这里初始化 `uni-app` 项目骨架。
